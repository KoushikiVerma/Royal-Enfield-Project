# Royal Enfield Web Application

## 🚀 Project Overview
This is a full-stack web application developed using Flask and MySQL for managing bike bookings and services.

## 🛠 Tech Stack
- Frontend: HTML, CSS
- Backend: Python Flask
- Database: MySQL (XAMPP)

## ✨ Features
- User Signup/Login
- View Bikes
- Book Bikes (Full Payment / EMI)
- Service Booking
- Admin Dashboard
- Manage Bikes, Services, EMI
- View Bookings (Service + Bike)

## ⚙️ Setup Instructions
1. Install Python
2. Install dependencies:
   pip install -r requirements.txt
3. Start XAMPP (MySQL)
4. Import database (.sql file)
5. Run:
   python app.py
